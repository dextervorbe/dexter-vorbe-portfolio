# Dexter Vorbe Portfolio

A personal portfolio website for Dexter Vorbe, built with React and Vite to showcase recent web projects, contact information, and social links.

## Live Features

- Hero intro with animated typing effect
- Project showcase grid with live links
- Contact section with clickable email and phone actions
- Social links (LinkedIn, GitHub, Instagram)
- Responsive footer and mobile-focused layout tweaks

## Technologies and Languages Used

### Languages

- JavaScript (ES Modules)
- JSX
- HTML5
- CSS3

### Frontend Libraries and Frameworks

- React `18.3.1`
- React DOM `18.3.1`

### Build Tools and Dev Tooling

- Vite `5.x`
- `@vitejs/plugin-react`
- npm scripts (`dev`, `build`, `preview`)

## Project Structure

```text
src/
  App.jsx
  App.css
  index.css
  main.jsx
index.html
```

## Getting Started

### Prerequisites

- Node.js (LTS recommended)
- npm

### Install

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
